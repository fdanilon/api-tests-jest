# API Tests with Jest

# How to execute?
- Install [Node.js](https://nodejs.org/en)
- Clone the repo
- Open the terminal in the project and execute the following commands
    - ``` npm init -y ```
    - ``` npm install jest supertest ```
- Change the package.json to
  ```
  "scripts": {
    "test": "jest"
  },
  ```
- In the terminal run the command ``` npm run test ```
